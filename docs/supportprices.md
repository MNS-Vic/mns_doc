# 支持字符与价格

# 支持字符

目前支持**小写英文**和**数字**

# 价格

| 位数       | 开放状态 | 年费            |
| :----------: | :--------: | :---------------: |
| 10 位以上  | 未开放   | -               |
|**5-10位**| **已开放**|**$5 等值的 Xin**|
|**4位**|**已开放**|**$160 等值的 Xin**|
|**3位**|**已开放**|**$640 等值的 Xin**|
| 2 位及以下 | 未开放   | -               |
